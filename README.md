# Profit Margin Calculator

An App that calculate your profit margin value to whatever amount you put on the input

How do you calculate gross profit margin?

Itâ€™s simple to find gross profit margin automatically using the calculator. To calculate manually, subtract the cost of goods sold (COGS) from the net sales (gross revenues minus returns, allowances, and discounts). Then divide this figure by net sales, to calculate the gross profit margin in a percentage.
